## Featured Project

### [OpsCore — HR Operations Intelligence Dashboard](https://github.com/anushkanaidu/opscore-dashboard)

> End-to-end HR lifecycle dashboard with AI-powered risk detection, automated alerts, and named ownership for every breach.
